# SHL Assessment Recommendation Agent

## Project Description
A conversational agent for recommending SHL catalog assessments based on user queries.

## Folder Structure
- src/: Main source code (api, agent, catalog, llm, models, core, utils, evaluation)
- tests/: Unit, integration, api, and evaluation tests
- data/: Raw, processed, and vector index data

## High-Level Architecture
The architecture follows a strict finite-state machine with distinct deterministic components for analyzing intent, retrieving assessments, ranking, and ensuring JSON API schema compliance.
